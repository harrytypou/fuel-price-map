How it works

Built for Vercel

This version does not need a custom local Node server, D3 CDN, TopoJSON CDN, or browser-side scraping. The page is static, while Fuelo and currency requests go through Vercel serverless functions in the /api folder.

  • /api/fueloFetches Fuelo.eu in EUR and parses the country fuel table server-side.
  • /api/ratesFetches current exchange rates with EUR as the base currency.
  • Local SVG mapNo map CDN dependency, so the map always appears after GitHub → Vercel deploy.
  • FlagDownload flagsCountry flags are loaded as real PNG images from flagdownload.com.
