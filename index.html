Data & implementation notes

Fuel prices are requested from https://fuelo.eu/?convertto=eur. Currency conversion is requested from https://api.frankfurter.dev/v1/latest?base=EUR. The vector map is loaded from the public world-atlas TopoJSON package. Country flag images are loaded from FlagDownload using PNG URLs such as https://flagdownload.com/wp-content/uploads/Flag_of_Austria_Flat_Square-1024x1024.png.

Important: many data websites block browser-side scraping with CORS or bot protection. The included server.js file solves that by serving the page locally and proxying Fuelo through /api/fuelo. For deployment, replace the same route with a Netlify/Vercel/Cloudflare function.
